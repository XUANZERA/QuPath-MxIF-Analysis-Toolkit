# Output Schema

This document describes the exported CSV and GeoJSON formats generated by the workflow.

---

# CSV Exports

## export_cells_by_annotation_*.csv

Cell-level measurements grouped by parent PNI annotation.

| Column | Meaning |
|---|---|
| `Annotation_Class` | Parent PNI annotation class |
| `Annotation_ID` | QuPath annotation object ID |
| `Annotation_Name` | Parent annotation name |
| `Cell_ID` | QuPath cell object ID |
| `Classification` | Cell classification |
| `Object type` | Cell / Detection |
| `geojson_path` | Linked nerve-region GeoJSON file |
| `Centroid X µm` | Cell centroid X coordinate |
| `Centroid Y µm` | Cell centroid Y coordinate |
| `Distance to annotation with nerve_regions µm` | Distance to nerve region |

---

# GeoJSON Exports

## geojson_<annotation_id>.geojson

Contains nerve-region geometries associated with a parent PNI annotation.

---

# Project-level exports
